Project: The World of Oneiro

Description

Saige is a health and wellness, mobile web application that allows users to search for different home remedies based on pre-defined list of ailment categories.

Saige was built using Agile / Scrum method and modern JavaScript framework React. Along with ReduxJS, ES6, Sass, PHP, MySQL, and AWS.
